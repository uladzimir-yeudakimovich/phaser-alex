About

Some useful information.

Genre

It is a multi-level platform game in which the character must collect all the car balls to advance to the next level.

Technologies

This game is built using the following technologies:

JavaScript;

Phaser;

Web Audio API.

Resolution

Minimum screen resolution for proper display of the game is 1120x640.

Difficulty

The game for kids from 6 years but will also be fun for their parents.

Controls

It's very simple.

  • JUMP

    Up arrow

    Just hit the up arrow on your keyboard to make Alex soar up like a butterfly.

  • GO TO THE RIGHT

    Right arrow

    Just hit the right arrow on the keyboard to make Alex start moving to the right.

  • GO TO THE LEFT

    Left arrow

    Just hit the left arrow on the keyboard to make Alex start moving to the left.

  • RESTART

    Down arrow

    Just hit the down arrow on the keyboard to restart game.
